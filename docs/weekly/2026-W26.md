## 本週目標

上週已完成 `DINOv3: supervised image-level AD on VisA` 的 CLS-linear baseline。本週不再把主軸放在 encoder selection，而是固定 DINOv3，評估 downstream anomaly component：patch-token scoring、Top-K aggregation、ViT projector。

## 已完成事項

- 實作 `patch_topk` feature mode：從 DINOv3 patch tokens 產生 patch-level logits，使用 Top-K=6 aggregation 得到 image-level anomaly score。
- 實作 `patch_vit_topk` feature mode：在 patch tokens 後加入 6-layer ViT projector，再做 patch-level head 與 Top-K aggregation。
- 完成 `EXP-002`：DINOv3 frozen + patch linear head + Top-K=6。
- 完成 `EXP-003`：DINOv3 frozen + 6-layer ViT projector + patch head + Top-K=6。
- 已產出 overall 與 per-category image-level metrics，並整理成 GitHub Pages 表格。
- 補上 `EXP-004/005`：token-wise MLP projector 與 local Conv projector controls。
- 補上 `EXP-006`：Transformer projector Top-K sweep (`K=1/6/12`)。
- 新增 patch-grid localization proxy，檢查 image-level 分數是否對應到瑕疵區域。

## 技術決策

- Encoder selection 暫時定案為 DINOv3，不再把主要進度放在 DINOv3 vs. other encoders。
- 後續實驗主軸改成 DINOv3 feature usage 與 anomaly-specific downstream module。
- `EXP-002` 是 patch-token Top-K control：先確認不用 CLS、改用 patch-level evidence 後是否有收益。
- `EXP-003` 在同樣 DINOv3 / VisA / Top-K protocol 下加入 ViT projector，結果顯示 token interaction module 是目前最有價值的 downstream component。
- `EXP-004` 顯示 MLP projector 已解釋掉大部分提升，因此「多了一個 nonlinear projector」本身就是重要因素。
- `EXP-005` 顯示目前的 local Conv projector 不是好方向，尤其 patch-grid localization proxy 幾乎失效。
- `EXP-006` 顯示 `K=6` 在 `1/6/12` 初步 sweep 中最好，但尚不能宣稱 global best。

## 本週可報告的研究價值

- `EXP-001` 建立 frozen DINOv3 在 VisA image-level supervised AD 的可重現基準。
- `EXP-002` 顯示直接用 patch token + Top-K 只有小幅收益，不能只靠 spatial patch evidence 解決問題。
- `EXP-003/004` 顯示 projector 是值得保留的 visual module；Transformer 略優，但 MLP 已很接近，下一步要做 parameter-matched control。
- 目前 claim 應限制為 image-level component baseline；尚未完成 pixel-level localization、parameter-matched control 或完整 FoundAD-style objective。

## 風險與阻塞

- 目前仍是 image-level supervised metric，尚不能宣稱 pixel-level localization 完成。
- patch-grid localization 是 14x14 proxy，不等同 full-resolution pixel metric。
- MVTec official protocol 不適合直接套目前 supervised VisA head；下一步應做 normal-only memory-bank decoder。
