## Findings

- 目前不能說「加 projector 就理所當然更好」。depth9 MLP 沒有贏 depth2 MLP，代表參數量不是主要解釋。
- depth2 Transformer image AUROC 最高，但 dense F1max 明顯低於 6-layer Transformer，表示 image-level 與 localization 可能拉扯。
- 6-layer Transformer projector 仍是目前最平衡的 supervised DINOv3 visual-module baseline。
- Normal memory-bank 在 VisA image-level 弱，但 dense localization proxy 強，適合作為 MVTec / pixel-level 路線的 baseline。
- 本輪仍沒有對齊博士班簡報 P13-P14 的 `99` 分 supervised oracle result；目前只對齊 task/dataset，不對齊 full Anomaly-OV pipeline。

## 風險與限制

- Dense heatmap 是 `224x224` proxy，不是原始解析度 full pixel metric。
- Memory-bank 目前是 single global bank，尚未做 class-specific bank、coreset、neighborhood restriction 或 morphology。
- MVTec image AUPRC 受 anomaly 比例影響，不能單獨當主要 claim。
