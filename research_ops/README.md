# Internal Research Operations

This directory is the internal working layer. It is not part of the MkDocs public site.

Use it for:

- raw lab discussion notes
- meeting notes before sanitization
- private TODOs
- report drafting before public cleanup

Public-safe summaries should be copied into `docs/` only after removing sensitive or unverified content.

## Directory Layout

| Path | Purpose | Git policy |
| --- | --- | --- |
| `internal_discussions/` | raw or detailed lab discussion notes | contents ignored except `.gitkeep` |
| `raw_notes/` | scratch notes, pasted references, temporary drafts | contents ignored except `.gitkeep` |
| `templates/` | reusable local templates | tracked |
