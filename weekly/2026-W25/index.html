    2026-W25 週進度¶

    日期：2026-06-15 至 2026-06-21

    本週目標¶

    建立可由 Git push 自動更新的研究進度網站，並把 DINOv3 baseline 的工作流拆成可追蹤的文件、決策與實驗紀錄。

    已完成事項¶

    • 建立 MkDocs 公開研究進度網站。
    • 將公開摘要與內部原始討論紀錄分離。
    • 建立第一版 12 週研究規劃，主軸為 DINOv3 supervised anomaly detection baseline。
    • 將 VisA 資料複製到 DINO work directory。
    • 完成 DINOv3 feature extraction smoke test，確認 CLS token 與 patch token shape。
    • 建立第一個 supervised baseline 訓練入口。
    • 加入 Mermaid 圖表，用於研究 pipeline、12 週規劃與文件更新流程。
    • 加入 MathJax 支援與指標定義頁，方便後續週報、國科會報告與 paper 使用一致符號。
    • 提交 EXP-001 到 Slurm。第一次 job 237545 找出 sbatch repo-root 問題，已改成使用 SLURM_SUBMIT_DIR。
    • 重新提交 EXP-001，Slurm job 237546 已在 hgpn18 成功完成，耗時 00:05:57。
    • 新增 image-level per-category evaluator，可輸出 P14-style category table 與 per-image prediction dump。
    • Slurm job 237589 已完成，用現有 EXP-001 checkpoint 重跑完整 test set evaluation，耗時 00:00:37。

    實驗進度¶

    • EXP-000 已完成：DINOv3 feature extraction 回傳 CLS (1, 768) 與 patch tokens (1, 196, 768)。
    • EXP-001 已完成：VisA 2cls_highshot.csv 上的 frozen DINOv3 + linear head baseline。
    • 初步結果：AUROC 0.9099、AUPRC 0.6915、F1 @ 0.5 為 0.5395、best F1 為 0.6176，對應 threshold 0.6364。
    • EXP-001 per-category image-level table 已完成，job id 為 237589。
    • tiny CPU training-loop smoke 已通過，但只代表程式路徑正確，不作為有效模型結果。

    技術決策¶

    • 第一個視覺 encoder baseline 使用 DINOv3，先固定 encoder，只訓練 anomaly head。
    • 第一階段先做 VisA image-level supervised anomaly detection。
    • 完整 VLM reasoning、Anomaly-OV reimplementation 與多資料集訓練，等乾淨 baseline 成立後再進行。
    • GitHub Pages 僅發布可公開摘要，不放內部原始討論。

    風險與阻塞¶

    • DINO 需要設定正式 Git remote 後，GitHub Pages 才能自動部署。
    • GitHub repository settings 需要選擇 GitHub Actions 作為 Pages source。
    • 若 repo 對外公開，push 前必須確認 docs/ 沒有內部敏感資訊。
    • CPU 上 DINO forward 太慢，正式 smoke 與訓練應在 GPU node 上執行。
    • 目前可以宣稱「第一個 supervised image-level baseline 可運作，且已完成 overall 與 per-category 指標」；下一步仍需 failure-case inspection。

    下週計畫¶

    • 執行 EXP-002：比較 MLP head 與 linear head。
    • 檢查 EXP-001 中 AUPRC / F1max 較低的類別，優先看 macaroni1、macaroni2、pcb2。
    • 規劃 pixel-level baseline，但不混入目前 image-level classifier。
